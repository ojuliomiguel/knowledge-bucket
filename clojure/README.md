# Clojure Study Notebook

> A man on a thousand mile walk has to forget his goal and say to himself every morning, ‘Today I’m going to cover twenty-five miles and then rest up and sleep.

A Clojure study repository organized by learning track. The early `the-fool` project was folded into the main tree; the name came from the idea that every journey starts small.

## Structure

- `src/brave_clojure/`: chapter-based studies, concept explorations, and early language experiments.
- `src/exercism/`: exercises from learning platforms, mainly Exercism.
- `src/katas/`: self-contained practice problems and small authorial exercises.
- `slip_box/`: notes, explanations, and exercise lists.
- `archive/calva/`: Calva onboarding material kept for reference, outside the main study flow.

## Usage

- Start a REPL with `clj`.
- Load files individually in the editor or with `require`.
- Use `slip_box/exercises.md` as a practice index.
